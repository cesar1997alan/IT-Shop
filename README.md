# IT-Shop - E-commerce (HTML + CSS Grid)

Proyecto de práctica: maqueta de e-commerce usando HTML y CSS (Grid).

## Demo
- Sitio: https://it-shoop.netlify.app/

##  Tecnologías
- HTML
- CSS (Grid)

##  Secciones / Páginas
- Inicio (`index.html`)
- Nosotros (`nosotros.html`)
- Producto (`producto.html`)

##  Cómo ejecutarlo
1. Clona este repositorio
2. Abre `index.html` en tu navegador
   - (recomendado) usa Live Server en VS Code
